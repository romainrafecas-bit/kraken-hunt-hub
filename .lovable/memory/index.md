# Project Memory

## Core
- SaaS for Cdiscount product velocity tracking via reviews. AliExpress/Lens sourcing.
- Routing: Landing page at `/`, dashboard at `/dashboard`. Catch-all redirects to `/`.
- Theme: Deep Abyssal Dark. Navy bg, Teal primary, Mint accents. High contrast. No orange/coral.
- Fonts: Nunito (headings/stats), DM Sans (body). No serif/monospaced.
- UI: Lucide via `GlowIcon` (bioluminescent hover), `rounded-lg`. No standard emojis.
- Lexicon: 'Récurrences' (sales), 'Vendeurs' (trends), 'Zones de chasse', 'Rejoins l'équipage' (CTA).
- Performance: Lazy load Recharts. `useInView` for animations. No infinite glows.
- Stack: Supabase (waitlist table with RLS).
